 - Solution implementation -
1. The migration process.
Есть четыре фазы для миграции. Это не раздельные фазы, а итерационный процесс. AWS предоставляет комплесную тулу чтобы облегчить каждый степ.
Assessment phase.
Определите готовность кастомера к работе в облаке. И потенциальный выхлоп от миграции.
Readiness and planning. (готовность и планирование)
Устраните пробелы, которые были у кастомера в предыдущей фазе.
Оцените environment кастомера и определите стратегию миграции (rehost, replatform).
Set up a secure and well-architected multi-account AWS environment - AWS Landing Zone.
Migration.
Из портфолио в индивидуальное приложение.
Каждое приложение разработано, мигрировано и валидировано.
Автоматически или вручную смигрировать в амазон.
Миграция данных.
Operations and Optimizations.

2. Minimum Viable Product (MVP) (Минимальный Жизнеспособный Продукт)
Это когда кастомер видит прогресс выполнения.
Это функциональный продукт достаточный, чтобы удовлетворить кастомера. И оставить место для фидбэка для будущей разработки решения.
Позволяйте кастомеру держать в уме новые идеи. Избегайте "big bang" решений, которые имеют ценность только в конце цикла.
Начните с основ и собирайте фидбек по мере того, как становитесь всё более сложным.
3. MVP and delivering results.
MVP-это разница между получением ценности здесь, в самом начале проекта, и получением ценности на разных стадиях принятия всего проекта.
POC - быстрая демонстрация решения. MVP - production level.

3. Containers.
Package code, configurations and dependencies into a single object.
Share an operating system
Run as resource-isolated processes
AWS offers resources and orchestration services
Use cases:
- microservices
- batch processing
- machine learning
- hybrid applications
- application migration to the cloud
- platform as a service
4. Analytics and data lakes.
Data in different silos can be difficult to access and analyze
Store data in a "data lake"
Easy to read data and abtain insights
Services:
- data movement: import your data from on premises and in real-time
- data lake: store any type of data securely, from gigabytes to exabytes
- analytics: analyze your data with the broadest selection of analytics services
- machine learning predict future outcomes and prescribe actions for rapid response
Все данные в одном месте. Это быстрее, легче и лучше анализировать.
Value:
- improved customer interactions
- improved R&D innovation choises
- increase operational efficiencies
